# Cloudinary Kurulum Rehberi

Resim yükleme işlemini ücretsiz ve kredi kartsız yapmak için Cloudinary kullanacağız. Lütfen aşağıdaki adımları takip edin:

## 1. Hesap Oluşturma
1.  [cloudinary.com](https://cloudinary.com/users/register/free) adresine gidin.
2.  **Sign Up for Free** butonuna tıklayın.
3.  Google hesabınızla veya email ile ücretsiz üye olun.
4.  "Welcome" ekranında size sorulan soruları rastgele seçip geçebilirsiniz (Role: Developer, vb.).

## 2. Cloud Name Bilgisini Alma
1.  Hesabınız açıldığında **Dashboard** (Ana Ekran) açılacaktır.
2.  Sol üst köşede **"Product Environment Credentials"** kutusunu göreceksiniz.
3.  Orada yazan **Cloud Name** bilgisini kopyalayın ve bir yere not edin.  bizimki :  dmfnsj2gz

## 3. Upload Preset Oluşturma (Önemli!)
Mobil uygulamadan şifresiz yükleme yapabilmek için "Unsigned Preset" oluşturmaniz gerekiyor:

1.  Sol menüden **Settings (Çark ikonu)** -> **Upload** sekmesine tıklayın.
2.  Sayfayı aşağı kaydırıp **"Upload presets"** kısmını bulun.
3.  **"Add Upload Preset"** linkine tıklayın.
4.  **Signing Mode** kısmını: **"Unsigned"** olarak değiştirin.
5.  **Save** butonuna basın.
6.  Listede yeni oluşan "Name" sütunundaki değeri kopyalayın bizimki :  gcpbapyz
